Beer Cat
Targeting the craft beer and tourism industries, Beer Cat is a 'React on Rails' web application that invites users to experience personalized tours of breweries and craft beers. By gathering individual preferences, the app determines a percentage likelihood that each registered user would enjoy the particular beers on tap at any given brewery.
My Focus: Designed ERD, coded user sign-up and authentication, built wireframes, logo, color scheme, and overall styling Tech Stack: React, Rails, PostgreSQL, Google Maps, Bootstrap, and HTML/CSS/SCSS.

Jungle
Welcome to the Jungle! Jungle is a mini e-commerce application allowing users to add items to their cart and checkout using Stripe. Guests are able to see Average Ratings and read Reviews from Users. Users can signup, login, and review products. Users can then delete reviews, but only the ones they have posted.
My Focus: From inherited code, I found and resolved bugs, implemented cart and checkout functionality, added user ratings, average ratings, and user authentication Tech Stack: Ruby on Rails, ERB, PostgreSQL, Stripe, Bootstrap, and HTML/CSS/SCSS.
