PROJECTS

Jungle
Welcome to the Jungle! Jungle is a mini e-commerce application allowing users to add items to their cart and checkout using Stripe. Guests are able to see Average Ratings and read Reviews from Users. Users can signup, login, and review products. Users can then delete reviews, but only the ones they have posted.
Built using Ruby on Rails, ERB, PostgreSQL, Stripe, Bootstrap, and HTML/CSS/SCSS.

Chatty Cantina
A single-page chat application similar to Slack, Chatty allows multiple users to connect and share images and messages with one another. As a Star Wars fan, I chose to inject 'the force' into my styling.
Built using ReactJS, JSX, Express, Webpack, Babel, WebSockets, Node, and HTML/CSS/SCSS.

Where's The Fun? Decision Maker
Using ranked voting, WTF? helps groups of friends make decisions. Create a poll, provide options, and let friends vote via drag & drop using the unique share URL. The poll creator monitors scores through a unique admin URL that ranks options by popularity.
Built using JavaScript, Express, PostgreSQL, AJAX, NodeJS, Knex.js, and HTML/CSS/SCSS.

Tweeter
A simple, single-page Twitter clone that allows users to add new tweets and view existing.
Built using JavaScript, Express, MongoDB, jQuery, AJAX, NodeJS, and HTML/CSS/SCSS.
